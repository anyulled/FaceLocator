# T20 — Create minimal operator runbook

## Objective
Document how to operate the POC safely.

## Required sections
- prerequisites
- AWS profile usage
- Terraform apply/destroy
- Lambda packaging
- where logs are found
- how to inspect S3 objects
- how to remove data for a user/event
- reminder not to use the root user

## Acceptance criteria
- a new operator can deploy and inspect the POC from the runbook
- destructive actions are called out clearly
